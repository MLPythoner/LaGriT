Example 9: random distribution of points in a box

The objective is to create a point distribution within a rectangular region using the createpts/random/xyz command.

First a rectangular geometry is created by specifying the region inside a cube. The createpts/random/xyz command is then used to create a point distribution with a specified target spacing. The output consists of one gmv file.

Input     

lagrit_input_createrandom
