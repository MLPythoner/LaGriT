LaGriT Registration

Supported platforms are Mac OSX, Solaris, Linux. LaGriT does not run on Windows or DOS. An exacutible is distributed that does not require compiling or building. If one wishes to write their own user modules, it is suggested that you use the Absoft (C, F77, F90) compiler. If you use another compiler and have problems, we will not be able to help.

To obtain LaGriT fill out a the web form (link below) and instructions for downloading LaGriT will be emailed to you.
