Example 3: addmesh/append

The objective is to join two meshes to create a third using the addmesh / append command.

Two cubes, one smaller than the other, each having a different grid resolution are combined. The append option concatenates two meshes where imt, icr, and itetclr
of mesh2 are given the values max(imt(mesh1)).
