Example 2: createpts in a cylindrical geometry
The objective is to add points to a cylindrical mesh using the createpts command.

A cylindrical geometry is defined. The createpts/rtz command is used to create a point distribution within the geometry exercising various options available to the command. The output consists of one gmv file.

 
Output Image
cylindrical grid

Input file
*TEST createpts/rtz (lagrit_input_creatertz)
cmo/create/3dmesh
surface/outer/reflect/cylinder/0,0,0/1,0,0/1/
region/r1/ le outer /
mregion/m1/ le outer /
createpts/rtz/4,6,11/0,0,0/3,45,10/1,0,1/
dump/gmv/output_creatertz.gmv
* begin compare here
cmo/status
cmo/printatt//-all-/minmax
quality
finish
