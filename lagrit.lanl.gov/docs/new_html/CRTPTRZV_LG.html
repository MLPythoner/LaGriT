CREATEPTS/VECTOR
This routine is used to ratio zone the region of space spanned by the input number ni of copies of the input vector vij away from the initial point v0j using the desired coordinate system. The j-th component of the i-th vector Vij is reduced by rij at each step in the i-th direction away from the initial point. No attempt is made to insure that the 3 vectors are independent.
FORMAT:
createpts/vector/[ |xyz|rtz|rtp /
                [ n1,n2,n3
                 /v11,v12,v13/v21,v22,v23/v31,v32,v33
                  /v01,v02,v03
                 /r11,r12,r13/r21,r22,r23/r31,r32,r33 ]

default = 0:      ni, vi, v0j
default = 1:      rij
