Example 1: test using rmmat to modify a mesh.

The objective is to use the rmmat command to modify a mesh by removing one of the material regions.

The output consists of five gmv files, one showing the single material region, one showing a mesh with two materials included, and one each for the results of rmmat/node, rmmat/element, rmmat/all.

Input

lagrit_input_rmmat

Images of GMV input and output

Mesh with single material
rmmat / element

Mesh with two materials

rmmat / node

rmmat / all
