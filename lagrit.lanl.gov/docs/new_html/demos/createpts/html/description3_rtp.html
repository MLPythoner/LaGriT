Example 3: creatpts in a spherical geometry
The objective is to add points to a spherical mesh using the createpts command.

A spherical geometry is defined. The createpts/rtp command is used to create a point
distribution within the sphere. The output consists of one gmv file.

 
Images of GMV output
spherical geometry
 
Input file
*TEST createpts/rtp (lagrit_input_creatertp)
cmo/create/cmo1/
surface/s3/reflect/sphere/0.,0.,0./1.,1.,1./
region/r3/ lt s3 /
mregion/mr3/ lt s3 /
createpts/rtp/2,6,30/0,0,0/1,45,360/1,1,1
dump/gmv/output_creatertp.gmv/
