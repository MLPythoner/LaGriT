read/gocad
read an ascii GoCad TSURF file.  This command requires that the mesh object name be specified.  This file format contains no geometry information. See http://gocad.ensg.inpl-nancy.fr and http://gocad.ensg.inpl-nancy.fr/IMG/papers/Fileform.fm.htm for further details on GoCad.
FORMAT:
read/gocad/file_name/mesh_object_name
EXAMPLES:
read/gocad/file.ts/cmo1
